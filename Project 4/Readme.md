Create a simple Scene with AI:<br/>
  
3 (or more) Enemies with basic AI<br/>
Each should have a different behavior such as: walking, patrolling, jumping, shooting, etc.<br/>
The player must be able to get rid of the enemies such as: jumping on them, shooting, etc.<br/>
If the player touches an enemy, gets shot, etc. show text “Game Over”<br/>
If the player gets rid of all the enemies, show text “You Win” <br/>
You can use whatever graphics/theme (or change the text) as long as you meet the requirements.<br/>

Winning condition: press the button<br/>
Losing condition: touch an enemy<br/>

Keyboard Input:<br/>
Jump - Space; Move left - Left; Move right - Right<br/>
